HELPME-

pour executer le programme:
	- Ouvrir un terminal dans le dossier bin et executer en premier la commande pour inicialiser le serveur. 
	"java server.MainServer".
	En seconde, lancer un nouveau terminal, tousjours dans le dossier bin et inicialiser le client avec l'adresse du serveur.
	"java client.MainClient localhost 8000".
